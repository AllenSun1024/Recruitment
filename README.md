# 量化招聘 | Support IT Manager

**职位名称**: Support IT Manager（IT运维主管）

**职位类型**: 全职

**团队方向**: 企业IT基建与内部技术支持

## About the Role

我们正在寻找一位兼具执行力与服务意识的 Support IT Manager，负责公司 IT 基础设施运维、办公内网搭建与日常技术支持。你将保障公司核心办公环境稳定、安全、高效运行，并推动 IT 运维规范化与流程化建设。

## Responsibilities

- 负责公司IT基础设施日常运维（网络、服务器、终端、办公系统）
- 规划并搭建/优化公司内网架构（网络分区、权限策略、访问控制）
- 管理账号与权限体系（入离职开通、最小权限、审计留痕）
- 建立监控告警与故障响应机制，处理突发问题并推动根因修复
- 制定并执行IT运维制度、SOP、应急预案与安全基线
- 对接外部供应商（机房、网络、硬件、软件服务商），保障交付质量与成本可控

## Requirements

- 本科及以上，计算机、网络工程、信息安全等相关专业
- 熟悉Windows/Linux日常运维、网络基础（TCP/IP、VLAN、VPN、ACL）
- 具备内网搭建、网络排障、终端管理与权限治理经验
- 具备较强的跨团队沟通能力与服务意识，可独立推进问题闭环

## Preferred Qualifications

- 有量化、互联网公司IT运维经验
- 熟悉云平台与虚拟化技术
- 有脚本自动化能力（PowerShell / Python / Shell）

## What You’ll Gain

- 主导企业IT基建与内网体系建设，形成可复用的运维方法论
- 扁平化组织架构，决策链路短，方案可快速落地
- 有竞争力的薪酬与长期成长空间
